# Installation
* 1 - clone repo 
* 2 - create a virtual environment and activate
*  - pip install virtualenv
*  - virtualenv envname
*  - envname\scripts\activate (source envname/bin/activate)
* 3 - cd into project 
* 4 - pip install -r requirements.txt
* 5 - python3 manage.py runserver
